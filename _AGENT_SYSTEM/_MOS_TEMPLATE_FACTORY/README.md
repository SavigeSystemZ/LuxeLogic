# MOS Template Factory

This directory contains factory-only automation for the Meta Operating System.

## Commands

- `build-source-manifest.sh`
- `validate-meta-golden-examples.sh`
- `smoke-prompt-bundle.sh`
- `smoke-installed-meta-repo.sh`
- `validate-mos-template.sh`
- `run-automation-lane.sh`

Nothing here is copied into installed target repos by the normal MOS install flow.
