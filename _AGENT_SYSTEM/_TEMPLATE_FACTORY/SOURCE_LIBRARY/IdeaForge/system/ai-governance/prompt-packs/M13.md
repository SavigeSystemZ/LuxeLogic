# Prompt Pack — M13: IdeaForge as MCP Server (optional)

## Objective
Expose IdeaForge capabilities to external AI hosts via MCP, with read-only by default and progressive scope elevation for mutating actions.

## Deliverables
- MCP server module (skeleton) and configuration template
- Tool/resource/prompt definitions aligned to mcp/IDEAFORGE_SERVER_SPEC.md
- Authorization + scope model (read-only baseline)
- Audit logging + correlation IDs for requests and scope elevation
- Tests: authz, scope enforcement, token non-passthrough, and redaction gating on export tools

## Guardrails
- Default configuration must expose ONLY read-only tools.
- Never store or log tokens.
- Never pass through tokens to upstream services.
- Add a kill-switch to disable all mutating tools at runtime.
