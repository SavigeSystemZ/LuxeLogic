# Asset Taxonomy

## Tag Prefixes
- `sector_` — business sector or crypto vertical (e.g., `sector_tech`, `sector_smart_contracts`)
- `industry_` — industry granularity (e.g., `industry_semiconductors`)
- `chain_` — underlying chain or protocol (e.g., `chain_ethereum`)
- `venue_` — trading venue or exchange (e.g., `venue_coinbase`, `venue_nasdaq`)
- `category_` — high-level asset category (e.g., `category_layer1`)
- `theme_` — thematic exposure (e.g., `theme_ai`)

## Usage
- Tags live in `asset_universe.yaml` under each asset.
- Use tags for exposure analytics, UI grouping, and reporting.
- Prefer lowercase and snake case; avoid spaces.
