# One Step Only

Scope the next response to a single step (or at most two tightly-related steps when safe and low-risk).

**Do this now:**
1. Restate the current goal or task.
2. Propose or execute exactly **one** clear step (outcome, scope, plan, implement, verify, report). If the step is large, split into labeled substeps (Step 4a, 4b, 4c).
3. Do not proceed to the next step in this response unless it is a trivial follow-up (e.g. run one verification command right after the edit).
4. End with: what was done, verification command(s), and the single next step for the user to request.
