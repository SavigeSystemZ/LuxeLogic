# Session start (alias)

**Same as `/session-start`.** Start a new Cursor session with context loaded and readiness confirmed.

Do this now:
1. Run the equivalent of `/load-context` (load master architectural systems per `AGENTS.md` or `docs/AI_CONTEXT_INDEX.md`).
2. Confirm `docs/CURSOR_NEXUS.md` is available (Skills, Commands, Subagents, Rules).
3. Confirm Rules are loaded (project rules in Cursor).
4. State briefly: "Session ready. Rules, Skills, Commands, Subagents available. Use `/` for commands."
5. If session logging is active, note "Session started; context loaded."

Quick ref: Commands = `/` in chat; Skills = apply when task matches (CURSOR_NEXUS); Subagents = "use X subagent"; Rules = automatic.
