# Meta Memory Rules

- Use `meta_system/context/` for durable repo-local memory.
- Use `META_WHERE_LEFT_OFF.md` for short resume instructions.
- Prefer concise, evidence-backed status over long narratives.
- If a validation command is the basis for a claim, record the command and result.
- Do not treat emitted host prompts or generated adapters as the source of truth; the canonical docs remain the source.
