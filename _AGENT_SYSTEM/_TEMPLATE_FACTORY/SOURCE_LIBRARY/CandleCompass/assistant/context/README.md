# Context Files

Use this folder to preserve continuity between sessions.

## Minimum updates after major work
- Add key outcomes to `CHANGELOG.md`
- Add next-step handoff notes to `WHERE_WE_LEFT_OFF.md`
- Update `CURRENT_STATUS.md` when runtime/test/ops state changes
- Add unresolved work to `../TODO.md`

## Writing style
- Keep entries factual and dated.
- Favor actionable bullets over long narrative.
- Record commands and artifacts only when they matter for repeatability.
