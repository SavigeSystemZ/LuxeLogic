# Skill: Generate SSoT Blueprint (PRD/ARCH/DATA/NFR/RUNBOOK)

## Inputs
- Idea Intake Form filled
- Any existing constraints, competitors, and differentiators

## Steps
1) Summarize the idea in 2-3 sentences (problem, user, outcome).
2) Draft PRD with strict MVP scope and non-goals.
3) Draft Architecture as modular monolith with clear module boundaries.
4) Draft Data Model with entities, relationships, invariants.
5) Draft NFR focusing on security, privacy, observability.
6) Draft Runbook with local dev, tests, lint/typecheck, deploy/rollback.

## Outputs
- system/design-docs/PRD.md, system/design-docs/ARCHITECTURE.md, system/design-docs/DATA_MODEL.md, system/design-docs/NFR.md, system/design-docs/RUNBOOK.if2.md
- Each file includes acceptance criteria and validation references.
