# M9 AIAST System Review

## Startup

Load `META_AGENTS.md`, `meta_system/META_INSTRUCTION_PRECEDENCE_CONTRACT.md`, `meta_system/META_REPO_OPERATING_PROFILE.md`, `meta_system/META_LOAD_ORDER.md`, and `docs/ReviewWorkflow.md`.

## Objective

Review an existing AIAST-like system for conflicts, drift, missing validation, weak authority chains, adapter divergence, and unsupported claims.

## Output Shape

- findings first, ordered by severity
- open questions and assumptions second
- concise change summary only after findings
