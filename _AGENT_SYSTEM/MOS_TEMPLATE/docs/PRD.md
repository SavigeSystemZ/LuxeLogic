# Product Requirements Document

## Goal

Build a Meta Operating System that guides AI agents in creating and evolving a system-of-systems template for application-building repos. MOS is a system for building the system that later builds apps.

## In Scope

- canonical meta contracts and operating rules
- donor intake and source manifests
- golden examples and category guidance
- plugin framework and adapter generation
- prompt emission, validation, lint, repair, and hallucination detection
- a structured review workflow for auditing existing systems like AIAST

## Out of Scope

- application runtime code
- non-neutral donor runtime copies
- real plugin feature implementations beyond a sample skeleton
- downstream UI surfaces
