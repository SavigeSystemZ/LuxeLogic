# Context State

This folder holds durable context for the master-repo-only meta-system workspace.

## Files

- `CURRENT_STATUS.md` — current operating reality for the meta workspace
- `DECISIONS.md` — durable boundary and structure decisions for the system-for-the-system layer

## Rule

- Keep this focused on master-repo-only AIAST design state.
- Do not treat these files as installable app-repo context.
